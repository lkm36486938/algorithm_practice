# 퀵정렬

## 정의

-   기준 데이터를 설정하고 그 <b>기준보다 큰 데이터와 작은 데이터의 위치를 바꾸는 방법</b>입니다.
-   일반적인 상황에서 가장 많이 사용되는 정렬 알고리즘 중 하나입니다.
-   병합 정렬과 더불어 대부분의 프로그래밍 언어의 정렬 라이브러리의 근간이 되는 알고리즘입니다.
-   가장 기본적인 퀵 정렬은 <b>첫 번째 데이터를 기준 데이터(Pivot)로 설정</b>합니다.
